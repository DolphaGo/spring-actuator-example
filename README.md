# Spring Server monitoring

## 👉🏻 Goal
### Let's check health of our springboot server!

> ## 주요 Method
## /health

- 어플리케이션에 대한 기본적인 정보를 제공
- Status
    - Up : 서버 정상
    - Down : DB 연결 문제, 또는 디스크 공간 부족

## /metrics
- JVM memory 사용률
- CPU usage


## /loggers
- Application log 확인
- Runtime시점에 log level을 바꿀 수도 있음
